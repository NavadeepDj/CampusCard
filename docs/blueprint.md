# **App Name**: Campus Cart

## Core Features:

- ID Scan: Scan student/faculty ID to initiate purchase at any campus vendor.
- Inventory Display: Display real-time product availability for each vendor, e.g., '4 Samosas available'.
- User Accounts: Maintain user accounts for students and faculty to manage payment methods and purchase history.
- Vendor Listings: Provide a unified interface to view all vendors on campus.
- Transaction History: Enable transaction history viewable to the user.
- Real Time Stock Management: Enable real time stock managment. Keep the stock to be the same ammount with stock quantity shown to user in product avaiability feature

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) - A vibrant blue (#3399FF) to convey trust and efficiency.
- Background color: HSL(210, 20%, 95%) - A light, desaturated blue (#F0F8FF) to provide a clean and unobtrusive backdrop.
- Accent color: HSL(180, 60%, 40%) - A teal green (#33A3A3) to highlight key interactive elements, such as the scan button.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern, neutral, and readable design, suitable for both headings and body text.
- Use simple, clear icons for navigation and product categories.
- A clean, card-based layout with ample spacing to present information clearly and avoid clutter.
- Subtle animations on loading and interactions to enhance user experience without being distracting.
